# Friend Status Toggle

An interactive web project built using HTML, CSS, and JavaScript that demonstrates DOM manipulation, event handling, and dynamic UI updates.

## 🚀 Features

* Toggle between "Friend" and "Stranger"
* Button click interactions
* Dynamic text and style updates
* Smooth opacity transition effects

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ▶️ How to Run

1. Clone the repository
2. Open `index.html` in your browser

## 📌 Purpose

This project was created to practice JavaScript DOM manipulation and build interactive UI behavior.
